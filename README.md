### Selected Projects

**[TinyMyo](https://arxiv.org/abs/2512.15729)** ‚Äî 3.6M-parameter EMG foundation model via self-supervised masked reconstruction. State-of-the-art on NinaPro DB5, UCI-EMG, EPN-612. Deployed on GAP9 microcontroller (0.785s ¬∑ 44.91 mJ).

**[Subjectivity Detection ‚Äî CLEF 2025](https://arxiv.org/abs/2507.11764)** ‚Äî Sentiment-enhanced transformer embeddings for objective/subjective news classification.

**[Whisper-TikTok](https://github.com/MatteoFasulo/Whisper-TikTok)** ‚Äî Automated video generation pipeline combining OpenAI Whisper, EdgeTTS, and FFmpeg.

**[Sexism Detection](https://github.com/MatteoFasulo/Sexism-detection)** ‚Äî Fine-tuned transformers for EXIST 2023 + SemEval EDOS, including zero/few-shot in-context learning.

**[Scalable WebApp](https://github.com/MatteoFasulo/ScalableWebApp)** ‚Äî Scalable WebApp with monitoring using Docker Swarm ¬∑ Ansible ¬∑ Traefik ¬∑ InfluxDB ¬∑ Grafana.

**[Versal Embedded AI](https://github.com/MatteoFasulo/Versal-C-Code)** ‚Äî Deep learning optimization on AMD/Xilinx Versal VCK190 for real-time inference.
